Painel de Log de Requisições

Conectando ao servidor...
⏬
?

Este painel exibe logs de requisições HTTP em tempo real.

Envie requisições GET ou POST para https://louren.co.in/request-logger/collect/{some-path}[/?somequery]

// Exemplo 1: GET

fetch('https://louren.co.in/request-logger/collect/myrequest?p1=v1&p2=v2', {
  method: 'GET'
})
    
// Exemplo 2: POST

fetch('https://louren.co.in/request-logger/collect/myrequest?p1=v1&p2=v2', {
  mode: 'cors',
  method: 'POST',
  headers: { 'content-type': 'application/json' }, // obrigatório
  body: JSON.stringify({ a: 'aa', b: 'bb' }),
})
